# Frontend

Next.js 16 app that displays character cards with filtering, search, and a 3D flip animation on hover.

## Setup

```bash
npm install
npm run codegen
npm run dev
```

> `codegen` requires the backend to be running — it introspects the GraphQL schema to generate typed hooks and documents.

App runs at `http://localhost:3000`.

## How it works

- **Filters** (status, gender, search) are synced to URL query params via [nuqs](https://nuqs.47ng.com/), so filtered views are shareable
- **React Query** handles data fetching with 1-minute stale time — filter changes trigger automatic refetches through the query key
- **Dark mode** respects OS preference on first visit, then persists the user's choice in localStorage
- Cards use a pure CSS 3D flip effect (no JS animation library)

## File overview

| File | What it does |
|------|-------------|
| `components/character-card.tsx` | Flip card with gender icons (inline SVG, no icon lib) |
| `components/character-list.tsx` | Grid layout + skeleton/error/empty states |
| `components/filter-bar.tsx` | Custom dropdown (no headless UI dep) + search |
| `components/search-input.tsx` | Debounced input (300ms) |
| `components/theme-toggle.tsx` | Dark/light toggle with animated sun/moon |
| `hooks/use-character-filters.ts` | URL-synced filter state |
| `gql/` | Auto-generated by codegen, don't touch |
